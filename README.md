# SQL Practice - 1

This repository contains my SQL practice covering basic to intermediate concepts.

## Topics Covered
- Database & Table Creation
- CRUD Operations
- Filtering (WHERE, IN, BETWEEN)
- Aggregations (SUM, COUNT, MAX, MIN)
- Joins (INNER, LEFT, RIGHT, FULL)
- Constraints
- Temporary Tables

This is part of my daily practice to strengthen SQL skills for Data Analyst roles.

## SQL Practice - 2 (Advanced SQL)

This practice focuses on advanced SQL concepts used in data analysis.

### Topics Covered:
- Window Functions:
  - ROW_NUMBER()
  - RANK()
  - DENSE_RANK()
- LEAD() and LAG()
- Handling NULL values:
  - ISNULL()
  - COALESCE()
- FIRST_VALUE() and LAST_VALUE()

This helped in understanding advanced querying techniques for analyzing trends, ranking data, and handling missing values.
